2 Часть: создаем репозиторий

зайдите в терминал и повторяйте все действия видео

1. Пройдите регистрацию на сайте GitHub

2. Кликните на зелёную кнопку со словом new в левом верхнем углу

3. Дальше напишите название репозитории и нажмите на зелёную кнопку в нижнем правом углу со словом creature repository

4. Зайдите в настройки 'обозначается шестеренкой' Перейдите во вкладку pages

5. Выберите root

6. Пере обновите страницу несколько раз и перейдите по ссылке которая появится сверху

7. Когда вы перейдёте посылки у вас может Отобразится что есть проблема, впишите в конце URL адреса папку src
